Технический долг → CSS-модули

Замена styled (emotion) → CSS-modules

// before — styled.ts (emotion)
- const Button = styled.button`
- background: ${props => props.primary ? '#4fffb0' : 'transparent'};
- padding: 8px 16px;
- `;

// after — CSS-модули
+ import styles from './button.module.scss';
+ const Button = (props) => (
+ <button className={cn(styles.btn, { [styles.primary]: props.primary })} {...props} />
+ );

Явная изоляция стилей · быстрее hot reload · нет runtime overhead emotion · легче статический анализ
